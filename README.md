# Optimisation_SGD_Proximales
Dans ce projet, je présente une étude comparative entre les différentes méthodes d'optimisation appliquées à un modèle de classification des tumeurs du sein et à un modèle de prédiction des prix de l'immobilier en Californie.

Pour l'étude breast cancer : 
  + Un rapport fixant le contexte de l'étude et mettant en valeur les résultats obtenus
  + Un notebook contenant tous les codes utilisés
  + Le dataset utilisé

Pour l'étude california hosuing prices : 
  + Le notebook contenant le code et les visualisations comparatives des différentes méthodes d'optimisations
  + Le dataset utilisé
